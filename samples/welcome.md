# 欢迎使用 MD 阅读器 👋

这是一个**轻量级**的 Markdown / HTML 预览工具，专为 iPhone 设计。

## 它能做什么

- 通过系统「文件」选择器打开 `.md` / `.html`
- 仿 Typora 的舒适排版
- 一键切换 **明亮 / 暗夜** 模式（自动记住你的选择）
- 可「添加到主屏幕」当作 App 使用，支持离线

## 排版示例

> 引用块长这样，适合放提示或金句。

行内代码 `const x = 1` 与代码块：

```js

  return `你好，${name}！`;
}
```
